# Project-1
NeuroEase: A Pain Management Device using Machine Learning
NeuroEase is an intelligent pain management platform that combines IoT sensors, machine learning, and cloud analytics to monitor physiological signals and assess pain levels in real time. The system utilizes a K-Nearest Neighbors (KNN) model to classify pain severity and automatically recommend adaptive therapy based on prediction results. The platform includes REST APIs for sensor data ingestion, cloud-based monitoring, and a visualization dashboard for healthcare professionals.

***Key Features***

Real-time physiological data collection

KNN-based pain level prediction

Adaptive therapy recommendation engine

RESTful API integration

Cloud analytics dashboard

Healthcare monitoring interface

Data visualization and reporting

***Tech Stack***

Python

Scikit-learn

IoT Sensors

React

REST APIs

Cloud Analytics
